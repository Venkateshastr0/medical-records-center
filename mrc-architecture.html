Request Data Flow
01
User Action
Staff interacts with React UI — search, form submit, or message send.
02
API / WS Call
Fetch or Socket.io emits request to Next.js server over LAN.
03
Zero-Trust Check
Middleware validates JWT, device fingerprint, IP trust score.
04
DB Query
API Route runs RBAC-gated SQLite query via business logic layer.
05
Decrypt PHI
AES-256 fields decrypted on-the-fly for authorized requester only.
06
Audit + Respond
Access logged to Audit Engine. Data returned to UI for render.

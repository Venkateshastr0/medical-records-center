# 🏥 Medical Records Center

## 📋 Project Overview

A comprehensive healthcare data management system with two implementations:

### **🪟 Original WPF Application**
- **Technology:** C#/.NET with WPF
- **Location:** `MedicalRecordsCenter/`
- **Status:** Complete desktop application

### **🌐 Modern Web Application**
- **Technology:** Next.js 14 + React 18 + TypeScript
- **Location:** `src-next/`
- **Status:** Complete web application with professional medical UI

---

## 🚀 Quick Start

### **Web Application (Recommended)**
```bash
cd src-next
npm install
npm run dev
```
Access at: `http://localhost:3001`

### **Desktop Application**
```bash
cd MedicalRecordsCenter
# Open in Visual Studio and run
```

## 🎯 Architecture

### **Web Application Stack**
- **⚛️ Frontend:** Next.js 14 + React 18 + TypeScript
- **🎨 Styling:** Tailwind CSS + Medical theme
- **📊 Charts:** Recharts for data visualization
- **🔐 Authentication:** JWT-based security
- **📱 Responsive:** Mobile-first design

### **Desktop Application Stack**
- **🪟 Framework:** WPF (Windows Presentation Foundation)
- **💻 Language:** C#/.NET
- **🏗️ Pattern:** MVVM (Model-View-ViewModel)
- **🗄️ Database:** SQLite
- **🔐 Security:** HIPAA compliance features
